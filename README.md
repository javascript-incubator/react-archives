# Curiosity

Public Portal to Library of Images gathered at NASA's Curiosity Mars Mission

[_Built with Chimera_](https://github.com/rajatsharma305/chimera)

### Environment Setup

* `npm install -g yarn`

### Project Setup

* `yarn install`

### Start Project

* `yarn start`

### Build Project

* `yarn build`
